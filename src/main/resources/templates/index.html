사내 문서를 이해하는 AI 챗봇 API

증권사 시연을 목표로 만든 Spring Boot 기반 AI 챗봇 서비스입니다. 금융 상품, 투자 기초, 시장 지표 문서를 바탕으로 질문에 답하고, 대화 기록과 피드백, 분석 기능으로 운영 흐름을 검증할 수 있습니다.

로그인하고 시작하기 계정 만들기
01 문서 기반 답변

업로드된 금융 지식 문서를 검색해 답변 생성에 활용합니다.

02 대화 관리

사용자별 대화 스레드와 질문, 답변 이력을 관리합니다.
